# 🚀 React + Tailwind CSS + Firebase Authentication

A modern frontend project built using **React**, **Tailwind CSS**, and **Firebase Authentication**. This project includes a clean UI, responsive design, and secure authentication system (Email/Password + OTP-ready structure).

---

## 📌 Features

* ⚛️ React-based frontend (Vite)
* 🎨 Tailwind CSS for styling
* 🔐 Firebase Authentication
* 📧 Email/Password login & signup
* 🔄 Auth state management
* 🚀 Fast development with hot reload
* 📱 Fully responsive UI

---

## 🛠️ Tech Stack

* React (Vite)
* Tailwind CSS
* Firebase
* React Router DOM
